# 🎓 Student Grade Analyzer

This is a beginner Python project that takes student name and marks as input and calculates the grade with a performance message.

## 🚀 Features
- Takes user input (name and marks)
- Validates marks (0–100)
- Calculates grade (A, B, C, Fail)
- Displays result with status message

## 🛠️ Technologies Used
- Python (Basics)
- VS Code

## 📌 How It Works
1. User enters name and marks
2. Program checks if marks are valid
3. Assigns grade based on marks
4. Displays final report

## 💡 What I Learned
- Python input/output
- Data types (int, string)
- If-elif-else conditions
- Basic project structuring
